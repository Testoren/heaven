## HEAVEN добро пожаловать на небеса!

### Данная работа предствленна скромным человеком с добрым сердцем, который всю зарплату отдает жене.

---

### Практическое задание по модулю 14.8.

Была разработа бета-версия сайта SPA салона "Небеса". Для корректной работы требуется выполнить следующие действия:

* Установить web сервер с поддержкой PHP и MySQL.
* Создать базу данных с именем "heaven" с кодировкой UTF (лучше всего utf8mb4_general_ci).
* Импортировать файл **heaven.sql** в созданную базу данных(например через phpMyAdmin).
* Распаковать все файлы из архива кроме **heaven.sql** в папку виртуального хоста веб сервера.
* Открыть файл `core/dbconf.php` и отредактировать переменную `$dbconf` указав параметры подкючения к СУБД MySQL.
